# Unity Scripts Pack
A Pack of a bunch of handy scripts for Unity *(may be a bit unorganised)*

##Contains
 - Bitmap Text Rendering
 - Audio Managers
 - Game Statistics
 - Very Simple Input Controller
 - UI Elements
 - IO Controller *(in dev)*
 - Open Project in Code Option
 - Utility Pack
 - Cheat System *(in dev)*
 - Pathfinding *(in dev)*
 - Min Max Drawer *(in dev but usable)*
